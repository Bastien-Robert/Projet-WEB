Présentation

Ce site internet vous permet de découvrir comment fonctionne un ordinateur et ses composants. Tout d'abord la page "Composants". Elle contient tous les composants indispensables au bon fonctionnement d'un PC :


  • Carte mère
  • Processeur
  • Carte graphique
  • RAM
  • Refroidissement
  • Stockage
  • Alimentation

La page "A vous de jouer", quant à elle, permet de créer votre propre ordinateur grâce à un formulaire.
Ensuite, la page "Boutique" contient tous les types de PC existant sur le marché actuel.
On continue avec la page "Actualités" qui nous affiche les meilleurs sites d'actualités PC.
Pour finir, la page "Qui sommes-nous" nous permet de rester en contact avec vous par le biais de nos réseaux sociaux.
